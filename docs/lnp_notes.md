# LNP 递送说明（第一阶段：文档占位）

## 1. 目的与边界

- 本文档仅做 LNP 递送的综述级记录，帮助阅读 `REPORT.md` 时理解上下游衔接。
- 当前仓库 **不生成** LNP 实际处方，不输出可直接用于制剂实验的配方文件。
- 第一阶段重点仍是：新抗原筛选、多价 mRNA 设计、SimHub 结构交付。

## 2. 常见文献名词（速览）

- 可电离脂质：如 SM-102、ALC-0315（不同体系与比例差异较大）。
- 辅助脂质：DSPC、胆固醇、PEG-脂质等。
- 关键工艺维度：N/P 比、粒径分布、包封率、稳定性与体内递送效率。

## 3. 与本仓库的关系

- 本仓库输出的是序列设计与计算证据，不直接推导 LNP 比例。
- 如果进入第二阶段制剂工作，建议新增独立配置文件（例如 `lnp_formulation.json`），并与 `scripts/run_all.py` 解耦，避免把湿实验参数硬编码进序列流程。

## 4. 报告中建议写法

- 在 `REPORT.md` 中可使用一句话说明：
  - “LNP 方案采用文献可复现路线（如 SM-102/ALC-0315 体系）作为后续实验输入，本仓库当前不自动生成配方参数。”
